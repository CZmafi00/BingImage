BingImage

BingImage is simple Api for retrieving Bing image of the day. Image of the day comes with various resolutions suitable for mobile and wide screens.

External dependencies

  • Newtonsoft - Json.NET (included in project)

Classes

  • BingImage
  • BingImageProperties
  • BingImageResolutions
  • BingImageUtilities

Example of usage

WebClient client = new WebClient();
client.Encoding = Encoding.UTF8;

string json;
string imageUri;

json = client.DownloadString(BingImage.BingApi);
BingImage image = Newtonsoft.Json.JsonConvert.DeserializeObject<BingImage>(json);
imageUri = image.GetUri();
image.SetResolution(BingImageResolutions.WideScreenRes_16_9._1366x768);

byte[] rawBytes = client.DownloadData(imageUri);
